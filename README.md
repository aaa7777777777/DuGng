# ⏱ 时间规划实时监测工具 / Task Time Planner

一个轻量级桌面端时间规划工具，专为**多任务并行推进**场景设计。

实时计时、进度追踪、剩余天数预估、每日打卡统计，帮你看清每个任务还需要多少时间，哪些部分必须用时间慢慢磨。

## 功能

- **任务分类管理** — 固定时间 / 深水区·细磨 / 日常推进 / 随机任务，四种类型颜色区分
- **实时计时器** — 一键开始/停止，自动累加时间，顶部实时显示
- **进度与预估** — 每个任务显示完成百分比、剩余小时、按每日投入预估完成日期
- **子步骤拆解** — 每个任务拆分子模块，标注预估时间，给出执行建议（整块/碎片）
- **子步骤打勾** — GUI 内直接切换子步骤完成状态
- **每日统计** — 查看今日各任务投入时间、目标达成情况
- **数据持久化** — 自动保存到本地 JSON，关闭重开不丢数据
- **导出报告** — 一键导出文本报告，含最近7天日志
- **深水区提醒** — 对需要长期细磨的任务给出心理预期管理提示
